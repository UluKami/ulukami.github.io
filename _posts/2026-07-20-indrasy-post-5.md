A slight progress report of sorts. The team I am working with at the IndraLabs Fellowship Program by Indrasy have started planning out all the features we want for our project through the form of user stories.  

I have seen these before during my time learning about brand work, but I haven’t ever applied these concepts to a project before. Sitting down and defining both features and how features relate to each other is a perfect way to be proactive. 

The reason I bring this up is because this step is a great example of how patience benefits a system. Originally, I had started planning the architecture that our service was going to use.  

In my haste, I accidentally forgot to design or even think about **how** a user is meant to use our service. If I hadn’t been made to take a step back and realize that the project was being rushed, we would have created a half-baked platform. 

Now that things are being taken step by step, the team and I are taking full advantage of patience. We have been attempting to apply the MECE (mutually exclusive, collectively exhaustive) framework when determining overlap between user stories, while ensuring that we touch every feature that we will need. 

By taking our time, we have already determined some user stories that we had missed when initially planning the architecture (my apologies, team). The hard part now is making sure that we continue to slowly but surely reach an ideal state using this framework. 

We are ultimately wrapping up this portion of the SDLC (software development life cycle) soon, so I don’t need to be so patient anymore (*lol*). I already have started coming up with ideas for how the architecture should be set up for our service and even have started playing around with reverse proxy servers to be able to accomplish some of the features we want to implement. 

I don’t want to sit on my hands while I wait after all, just taking it slowly but surely like I should have at the beginning. To sum it up though, I learned just how great the skill of patience can be.  

I am curious to hear how you have benefited from patience in a project (*or even in hindsight would have benefitted from patience lol*). Thank you for reading!
